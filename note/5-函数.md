## 函数

### 定义函数

在Python中，使用def来定义函数

```python
def my_abs(x):
    if x >= 0:
        return x
    else:
        return -x							
```

函数调用：

```python
my_abs(-9) 								#结果为：9
```

如果想定义一个什么也不做的空函数，可以使用pass语句：

```python
def nop():
    pass
```

pass也可以用在其他语句中用来做占位符：

```python
if age >= 18:
    pass
```
