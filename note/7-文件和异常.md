## 文件和异常

### 从文件中读取数据

```python
file_path = '/home/fileName.txt'
with open(file_path) as file_object:
    conent = file_object.read()
    print(conent.rstrip())
```

逐行读取：

```python
file_path = '/home/fileName.txt'
with open(file_path) as file_object:
    for line in file_object:
        print(line)
```

### 写入文件
