### 数字

在Python终端中，可直接返回数字运算结果，Python用两个*表示乘方运算

```
>>>2 ** 3					//结果为9
```

使用函数 str（）避免类型错误：

```
age = 23
message = "happy " + age + "birthday"					//错误
message = "happy " + str(age) + "birthday"				//正确
```
